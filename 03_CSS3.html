대소문자 설정

이번 태그는 영어 대문자와 소문자를 지정해주는 것을 말한다. 텍스트 속성을 설정하기 위하여는 'text-transform' 속성을 활용해주면 된다.

HTML은 Hyper Text Markup Language 입니다. = 영어 텍스트의 앞부분만 대문자를 사용한다.

HTML은 Hyper Text Markup Language 입니다. = 영어 텍스트를 모두 소문자만 사용한다.

HTML은 Hyper Text Markup Language 입니다. = 영어 텍스트를 모두 대문자만 사용한다.

HTML은 Hyper Text Markup Language 입니다. = 텍스트 변환 속성의 값이 부모 요소에서 상속하도록 지정한다.

HTML은 Hyper Text Markup Language 입니다. = 기본값으로 설정한다.

HTML은 Hyper Text Markup Language 입니다. = 찾아보니 뭔 속성을 초기화 시켜준다는대 사실 잘 모르겠다.

HTML은 Hyper Text Markup Language 입니다. = 아무런 효과 없다.
