1. q태그

짧은 인용구를 넣고 싶을 때, 사용하는 태그는 'q' 태그로, 태그를 사용하면, 따옴표가 자동으로 생긴다.
HTML은 Hyper Text Markup Language이다.

2. blockquote 태그

인터넷에서 사전적 의미를 정의할 때, 사용하는 표현을 하고 싶으면, 'blockquote'태그를 사용하면 된다.
HTML은

Hyper Text Markup Language
이다.

3. abbr 태그

'abbr' 태그는 대표적인 축약형 태그로, 마우스 오른쪽을 놓았을 때, 나오는 문장이다.

HTML은 HTML, CSS, Javascript 이다.

4. address 태그

'address' 태그는, 기울기로 주소를 표현하는 간단한 방법으로 태그명도 정말 간단하다.
